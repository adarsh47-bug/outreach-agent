## Contributing.md
We welcome community pull requests or private iterations:
1. Fork the repo and set up a custom workspace.
2. Initialize and deploy a unique Firebase Sandbox instance.
3. Validate tests and execute code quality checks:
   - Run `npm run lint` before committing files.
